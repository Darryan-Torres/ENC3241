# Instructions
How to create a file on a windows computer.


Step 1. Press the the windows button and search file explorer.
  
<img width="800" height="800" alt="Screenshot 2026-03-24 085945" src="https://github.com/user-attachments/assets/f9d18777-985c-4b95-b29c-b91abc080ae7" />



Figure 1 - Searching for file explorer (Screenshot by Darryan Torres)

Step 2. Open file explorer

Step 3. Open the folder where you want to make the new subfolder.

Step 4. Click the **NEW** button in the top left

Step 5. Click **Folder** in the submenu

<img width="267" height="383" alt="Screenshot 2026-03-24 at 9 14 11 AM" src="https://github.com/user-attachments/assets/2992f0bc-fa9e-493b-a4f1-e4522d794dd3" />



Figure 2 - Create new folder (Screenshot by Darryan Torres)

Step 6. Type a name for the new folder such as **instructions**

Step 7. Press the **Enter** key to save the folder
